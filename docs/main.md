# `main.py`

## Purpose
Minimal CLI entry point.

## Function reference

### `main()`
Prints a simple greeting (`Hello from rpi!`).

**Example**
```bash
python3 main.py
```

**Use case**
- Smoke-check that the Python environment starts correctly
- Packaging placeholder while the real dashboard lives in `tui.py`

## Notes
This file is intentionally tiny. It is useful as a sanity check, not as the primary launcher.
