# Spec: Complete Project Documentation Reference

## Goal
Produce a complete, detailed English documentation set for the Raspberry Pi Dumb TV Dashboard project, covering every Python function, UI flow, WebUI route, helper script, and test entry point with real usage examples.

## Requirements
- Keep documentation in English.
- Cover every function and class in the current codebase.
- Include real usage examples for the TUI, WebUI, API routes, keyboard daemon, and test scripts.
- Document the role of the Kodi tab, the Player tab, the Audio tab, and the Devices tab.
- Add a documentation index and update the main README to point to it.
- Preserve existing behavior; this is documentation-only unless a minor doc-linked text correction is necessary.

## Acceptance Criteria
- Documentation index exists and links all major docs.
- Each Python module has a dedicated reference section.
- Every public function/class/method is documented.
- Real-world examples are included for routine tasks.
- README points readers to the new docs.
